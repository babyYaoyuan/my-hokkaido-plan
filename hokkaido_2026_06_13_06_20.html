6/13-6/20 · 札幌为主 · 洞爷湖温泉一晚 · 不自驾

北海道轻松逛吃温泉攻略

这趟不做大环线，把时间留给札幌的小店、寿司与成吉思汗、小樽和余市、洞爷湖的湖景温泉。每天控制在轻松到适中，保留一个机动日给天气和心情。

每日执行卡

6/12 周五

杭州 → 上海浦东

低强度前置交通
晚间杭州下班后坐高铁到上海，直接去浦东机场附近酒店。
住宿选机场接驳或打车 15 分钟内到 T1/T2 的酒店。
准备护照、入境资料、日元现金、交通卡、机票放随身包。
避免住上海市区，6/13 尽量 05:30-06:00 到浦东机场。
6/13 周六

抵达札幌

低强度狸小路
上午08:20 浦东起飞。
下午12:50 到新千岁；JR 到札幌，入住后逛大通、狸小路。
晚上成吉思汗だるま / 札幌啤酒园 / 汤咖喱三选一。
入境或 JR 耽误时，直接酒店附近吃，不硬排名店。
6/14 周日

札幌祭 + 市区逛吃

中强度祭典
上午北海道神宫或中岛公园，看札幌祭摊位与仪式氛围。
午餐二条市场海鲜丼，或 Suage4 汤咖喱。
下午狸小路 1-7 丁目、Pole Town、Aurora Town。
人太多就改北海道大学、赤砖厅舍外观、札幌站商圈。
6/15 周一

余市 Nikka + 小樽

中强度海鲜
上午札幌出发去余市，Nikka Museum、商店、付费品鉴吧。
午餐小樽寿司或三角市场海鲜丼，别太晚到。
下午小樽运河、堺町通、北一硝子、音乐盒堂、LeTAO。
不想跑余市时，直接小樽一日慢逛。
6/16 周二

札幌深逛

中强度购物
上午北海道神宫例祭最终日巡游，留在市中心更稳。
下午札幌站、大丸、Stellar Place、PARCO、狸小路。
晚上预约成吉思汗、寿司或海鲜居酒屋。
如果巡游影响交通，全天只走步行可达路线。
6/17 周三

札幌 → 洞爷湖温泉

低强度温泉
上午札幌退房，大箱子寄存，只带一晚小包。
中午优先酒店接驳；备选道南巴士 11:30→14:10。
晚上酒店晚餐、泡汤，20:45-21:05 看洞爷湖烟花。
接驳满座就用直达巴士；下雨就酒店内休息泡汤。
6/18 周四

洞爷湖 → 札幌

低强度补购物
上午慢起早餐，再泡一次，10:00 接驳回札幌最省心。
下午入住札幌，札幌站商圈、地下街、百货地下。
晚上根室花まる、海鲜居酒屋或汤咖喱。
回程晚点时不要预约太早，百货地下熟食很稳。
6/19 周五

机动日

低-中强度天气决定
默认二条市场/场外市场，北海道大学，円山咖啡小店。
备选天气好报积丹海胆/神威岬一日游。
备选想坐观光列车，做富良野美瑛早夏花田。
最稳版本就是札幌慢逛 + 最后一顿认真吃。
6/20 周六

札幌 → 新千岁 → 浦东

低强度返程
上午09:30 退房，10:30 左右从札幌站坐 JR 去机场。
机场午餐、机场拉面道场、白色恋人、六花亭、Royce。
航班13:50 新千岁起飞，17:00 到浦东。
如果当晚回杭州，提前确认浦东到虹桥/杭州衔接。

住宿

札幌：大通 / 狸小路 / 薄野北侧

优先级酒店适合点注意
1Tokyu Stay Sapporo Odori房内洗烘机，长旅行友好确认房型是否带洗烘
2karaksa hotel Sapporo贴近狸小路，逛吃效率高轻睡眠者选高楼层
3HOTEL VISTA SAPPORO ODORI面向狸小路，位置直接房间可能偏紧凑
4Cross Hotel Sapporo位置居中，公共浴场加分价格可能超预算

洞爷湖：6/17 一晚

优先级酒店适合点交通重点
1Toyako Manseikaku价格、位置、接驳平衡札幌站北口 13:00 接驳，需预约
2The Lake View TOYA Nonokaze湖景和情侣感更强13:30→16:00，次日 10:00→12:30
3Grand Toya / Kanko Hotel预算友好备选优先湖侧房型
4Ko No Sumika / Sun Palace体验更强常超预算，看到特价再考虑

札幌尽量订同一家酒店前后两段，并确认能否 6/17 退房后寄存大箱子到 6/18 再入住。洞爷湖这一晚优先“一泊二食”。

交通速查

新千岁机场 ↔ 札幌

首选 JR Rapid Airport，到札幌约 30-40 分钟。新千岁机场 JR 站在航站楼 B1F。

时间6/13 建议动作
12:50抵达新千岁
13:40-14:20入境、取行李、到 B1F JR 站
14:00-15:00JR 去札幌
16:30 后大通、狸小路、薄野慢逛

札幌 ↔ 洞爷湖

优先酒店接驳。备选道南巴士直达 Toyako Onsen，成人单程 3700 日元，需预约。

方向班次建议备注
札幌 → 洞爷湖11:30 → 14:10不用酒店接驳时最合适
札幌 → 洞爷湖14:10 → 16:50睡懒觉版，到店偏晚
洞爷湖 → 札幌09:50 → 12:30回札幌吃午饭
洞爷湖 → 札幌12:50 → 15:30上午多泡一会儿

美食重点

成吉思汗

札幌代表性很强。だるま更经典，札幌啤酒园更宽敞稳定。

海鲜寿司

小樽、根室花まる、二条市场都可安排，优先扇贝、北寄贝、海胆、花咲蟹。

汤咖喱

Suage / GARAKU / RAMAI，适合午餐、雨天、疲惫日晚餐。

积丹海胆

6-8 月当季。6/19 天气好且愿意一日游时加入。

牛乳甜品

小樽 LeTAO、百货地下、新千岁机场都能补。

居酒屋

最适合吃刺身、烤鱼、玉米、土豆、当地酒，留给 6/18 或 6/19。

购物与小店

狸小路 1-7 丁目

古着、杂货、药妆、餐饮、小店。分两次逛更舒服。

Pole Town / Aurora Town

雨天路线，连接大通、薄野和狸小路。

札幌站商圈

Daimaru、Stellar Place、百货地下食品区，适合 6/16 或 6/18。

COCONO Susukino

新商场、餐饮、杂货，晚饭前后顺路。

NORBESA 周边

动漫、模型、娱乐、二手相关，适合找稀奇小东西。

小樽堺町通

玻璃、音乐盒、LeTAO、甜品和伴手礼。
